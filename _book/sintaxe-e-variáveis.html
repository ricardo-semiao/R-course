Capítulo 1 Sintaxe e Variáveis

1.1 Sintaxe

Imputando dados e operações:

  • Para imputar texto/strings, utilize " ou ': "Hello World!", 'Hello World!'
  • Para imputar números, simplesmente escreva-os: 5
  • Para imputar comentários (texto que não será “rodado”), use o símbolo #
  • Existem muitos operadores utilizados com símbolos especiais: 5+5, 5/2
    • Estudar a lista de operadores ficará de exercício
  • Parênteses podem ser utilizados em operações matemáticas: (5+5)/2

Expressões:

  • Expressões são excertos de código
  • Em vários casos especiais, utilizaremos chaves { para delimitar uma expressão: {5 + 5}
  • Podemos escrever mais de uma expressão na mesma linha usando ;: x <- 3; y <- 3; x + y #> 6

1.1.0.1 Definição

Por enquanto, uma função é uma expressão associada à um nome, podendo ser executada em diferentes contextos: 

nome <- {expr} #meramente ilustrativo
nome()

1.2 Variáveis

Operadores de definição:

  • Para definir variáveis, escrevemos seu nome, = ou <-, e a expressão que definirá seu valor: x = 3, x <- 3
    • = serve para mais coisas que somente definição de variáveis, como indicar argumentos em uma função, logo, <- funciona como “definidor” em mais contextos
    • <- serve de trás pra frente: 3 -> x, = não
  • Ambos podem definir várias variáveis de uma vez: x = y = 3, x <- y <- 3, x = y <- 3
  • Existe um terceiro operador <<-, que será discutido no tema “Enviroments”

Regras de nomenclatura:

  • Nomes podem conter letras, números, “.” e “_”
  • São “case-sensitive”
  • Podem começar apenas com letras ou “.”
  • Não podem ser palavras reservadas como “TRUE”
